备份与恢复

备份

# 备份单个数据库
shell> mysqldump -uroot -p123456 db_name > /backup/`date + %F`.sql
shell> mysqldump -uroot -p db_name > /backup/`date + %F`.sql  #这样输入密码不会出现在日志里，比较安全
shell> mysqldump -uroot -p -B db_name > /backup/`date + %F`.sql 
加了-B 参数后 恢复的时候就不需要先创建数据库了。

# 备份所有数据库:
shell> mysqldump -uroot -p  --all-database >all.sql

# 备份数据库test下的user表和passwd表
shell> mysqldump -uroot -p test user passwd> user.sql

恢复

shell> mysql -uroot -p db_name < backfile #db_name为导入数据库的库名，backfile为本地文件
如果是-B参数备份的化，恢复的时候就不需要db_name了
注意！ 是mysql 命令恢复 不是使用mysqldump
