socket编成

笔记
  • python
  • socket

socket 类型

代码 作用
socket.AF_UNIX 只能用于单一的Unix系统间进行通信
socket.INET 服务器之间的通信
socket.INET6 ipv6的通信
socket.SOCK_STREAM 流式socket, for TCP
socket.SOCK_DGRAM 数据包式socket, for UDP
socket.SOCK_RAW 原始套接字, 普通的套接字无法处理ICMP、IGMP等网络报文, 而SOCK_RAW可以；
  • 创建TCPsocket
import socket
tcpSocket = socket.socket(socket.AF_INET,socket.SOCK_STREAM)
  • 创建UDPsocket
import  socket
udpSocket = socket.socket(socket.AF_INET,socket.SOCK_DGRAM)

socket 函数

服务端

代码 作用
socket.bind(address) 将套接字绑定到地址, 在AF_INET下,以元组（host,port）的形式表示地址.
socket.listen(backlog) 开始监听TCP传入连接。backlog指定在拒绝连接之前, 操作系统可以挂起的最大连接数量。该值至少为1
socket.accept 接受TCP连接并返回（conn,address）,其中conn是新的套接字对象, 可以用来接收和发送数据。address是连接客户端的地址

客服端

代码 作用
socket.connect(address) 连接到address处的套接字。一般address的格式为元组（hostname,port）, 如果连接出错, 返回socket.error错误
socket.connect_ex(adddress) 功能与connect(address)相同, 但是成功返回0, 失败返回errno的值

公共函数

代码 作用
socket.recv(bufsize) 接受TCP套接字的数据, 数据以字符串形式返回, bufsize指定要接收的最大数据量
socket.send(string) 完整发送TCP数据。将string中的数据发送到连接的套接字, 但在返回之前会尝试发送所有数据。成功返回None, 失败则抛出异常
socket.recvfrom(bufsize) 接收UDP套接字的数据, 返回的是（data,address）
socket.sendto(string,address) 发送UDP数据, addres形式为（ip,port）
socket.close() 关闭套接字
socket.settimeout(int) 设置套接字操作的超时期, timeout是一个浮点数, 单位是秒
  • 强制终止socket程序, 由于bind了ip和端口, 重新运行会导致端口已被使用的报错。
  • 解决办法:复用端口 在bind之前添加如下代码。就不用等个2-4分钟了！
tcpCliSOck.setsockopt(socket.SOL_SOCKET,SO_REUSEADDR,1)
